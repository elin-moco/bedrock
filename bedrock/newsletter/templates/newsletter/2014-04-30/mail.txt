最新消息

[募集強大的 Web App 開發工具](http://blog.mozilla.com.tw/posts/5424/powerful-tools-for-developing-web-apps)
Mozilla 正集結核心工具，要推薦給 Web App 開發者，協助大家能順利的開發出作品。其工具將囊括目前 Web App 核心所用的工具鏈，詳細內容將公告於 MDN 上。
[我也想成為 mozillian！ Part 2 – 你也來Try Try 看](http://tech.mozilla.com.tw/posts/4243/)
Try server 是跑自動測試的地方，TBPL 則是用來顯示測試結果的工具，透過台客 jwwang 的介紹，你將對軟體開發的自動測試有進一步了解。
[Rormix – 透過 Firefox OS 找到正夯的音樂影片](http://blog.mozilla.com.tw/posts/5395/rormix)
Rormix 耗時一個月完成 iOS 與 Android 版本，並僅用「一天」的時間移植到 Firefox OS 上!快來看看這有趣的開發過程。
[懷抱未來的希望](http://myfirefox.com.tw/videos/21764)
Mozilla 投入行動事業，期望讓行動網路普及於民，世界上有更多人得以接觸到 web 的好。一同來看看 Mozilla 對未來的想像。
[Firefox OS App 系列影片 (3)：在模擬器中測試](http://blog.mozilla.com.tw/posts/5290/testing-in-the-simulator)
入門 Firefox OS 最簡單的方法為使用「應用程式管理員」，它能協助你安裝並測試開發中的 App，並即時觀看修改之後的結果。
[Firefox OS App 系列影片 (4)：在實機中測試](http://blog.mozilla.com.tw/posts/5328/testing-on-the-device)
實機測試可透過 Firefox 瀏覽器內建的「開發者工具」和「應用程式管理員」進行 App 即時編修，讓 App 測試更有效率。
