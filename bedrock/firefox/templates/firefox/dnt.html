全球性非營利組織 Mozilla 致力打造 Web 的美好未來。我們認為原則比利潤更為重要，也相信 Web 應該是開放並易於取用的全球公用資源，而非用以販售牟利的商品。我們將使用者放在第一位，並且深信每個人都應該可以自行控制 Web 的瀏覽體驗。針對你在上網時被第三方網站所蒐集、使用、儲存、分享的個人資訊，Mozilla 認為你有權利了解相關方式並能自己決定是否要被追蹤。

Mozilla Firefox 因此提供了「不要追蹤我 (Do Not Track，簡稱 DNT) 」功能，讓你透過瀏覽器告訴網站你不希望自己的瀏覽行為被追蹤。只要啟動此功能，Firefox 就會告知廣告性質的網站與應用程式，你拒絕受到如行為廣告等機制的追蹤。

常見問答集

什麼是 DNT？

不要追蹤我 (DNT) 可讓你控制本身資訊在網路上被蒐集與利用的情形。DNT 是 Firefox 提供的功能之一，可讓你告知網站你拒絕受到第三方追蹤，避免用於如行為廣告 (Behavioral advertising) 等目的。只要任一網站要求你的資料，此功能隨即會傳送 Do Not Track HTTP 標頭給予網站知道。

Firefox for Android 也有 DNT 功能嗎？

當然。Firefox for Android 是首款提供 DNT 隱私功能的行動瀏覽器，且運作機制與 Firefox 桌面版完全相同。若要啟動 Firefox for Android 的 DNT 功能，只要在瀏覽器中找出「設定」畫面，找到 "告訴網站我不想被追蹤" 的方塊並勾選之，即可開啟 DNT 功能。

DNT 也會封鎖廣告嗎？

不會。即使你啟用了 DNT 功能也還是看得到廣告，但 DNT 可能會更改你看到的廣告類型。舉例來說，行為廣告根據你所瀏覽的網站與搜尋的關鍵字，找出你感興趣的內容。一旦你告訴網站不想被追蹤，則網站就會轉而根據你的隱私性偏好，提供一般的普通廣告取代行為廣告。

DNT 搭配其他隱私工具的運作情形？

DNT 是眾多隱私解決方案之一，並不會取代防毒軟體、不會加密資料，本身也不屬於安全機制。Firefox 還具備許多其他隱私與安全功能。

DNT 會不會影響其他的 Web 經驗？

DNT 可能會干擾某些你已經習慣的個人化服務。舉例來說，每當你要看氣象報告時，DNT 就可能會要求你輸入郵遞區號一次，而不會像之前一樣自動顯示天氣狀況。網站的個人化機制雖然可幫你省時，讓你不需再重複鍵入資訊，但仍會請求某些特定資料。

如何開啟 Firefox 中的 DNT 功能？

Firefox 的 DNT 預設為關閉。你可在「隱私」面板中找到 DNT 請求。若使用 Windows，則可在 Firefox 中找到 工具 > 選項... > 隱私。

同樣勾選 "告訴網站我不想被追蹤" 的方塊即可。若需更多資訊，可參閱我該如何停止網站追蹤我？輔助說明。

這些公司會尊重我的 DNT 設定嗎？

許多公司都已經開始支援 DNT，但你現在可能還沒發現任何變化。不論是已經開始，或承諾會立即開始建構 DNT 的公司，Mozilla 都會積極接洽合作事宜。

哪裡可找到更多資訊？

有關 DNT 與線上隱私的有趣資訊：
  • Firefox 中的隱私瀏覽 功能 — 針對你瀏覽過的網站與頁面，均不會留下任何資料
  • Firefox 中的管理 cookies 功能 — 可讓你控制電腦儲存 cookies 的情形
  • Alex Fowler 的 Mozilla 部落格 — 由 Mozilla 技術與隱私總監提供最新消息
  • donottrack.us 專案 — 史丹佛大學正著手研究 DNT 背後的概念
  • 尚為 W3C 編輯狀態的 DNT 追蹤偏好表示式 (Tracking Preference Expression)
  • 尚為 W3C 編輯狀態的 追蹤規範與範圍 (Tracking Compliance and Scope)
  • W3C 追蹤防護工作團隊
  • Ghostery — 此工具可讓你知道目前正追蹤你的網站
  • 看看在你按下滑鼠之前，你的資料都去哪了 — 此圖是由 Future of Privacy Forum 所提供
  • Mozilla 與史丹佛大學的 DNT 提交至 IETF — IETF 屬於標準規範組織。不論你使用何種網路瀏覽器，均由 IETF 協助定義 Web 作業的方式
