Firefox

Mozilla 首次參與台灣校園徵才

  • 首次參與台灣校園徵才活動。
  • 在台灣大學、清華大學與交通大學三場活動中，吸引上千名學生與專業人才參與。

發表全新改版 Firefox

  • 正式發表 3 年以來全新改版的 Firefox。
  • 首度內建 Firefox Accounts，將 Firefox 帳號隨身帶著走。
  • 此一新版包含流暢且有趣新設計、新選單，與新自訂功能。

首度推出 Developers 版 Firefox

  • 為慶祝 Firefox 十歲生日，Mozilla發表首款專為開發者打造的瀏覽器。
  • 開發者版本可讓開發者對網頁進行除錯，包含如 WebIDE 與 Firefox Tools Adapter 的強大新工具。

Firefox Hello 問世

  • WebRTC架構的視訊＼音訊功能「Firefox Hello」現身。
  • 使用者不須另外下載軟體或建立帳戶，也能撥打免費語音或視訊電話。

台灣與全球 Firefox 歡慶十歲生日

  • Firefox 紀念版與開發者版上線。
  • Firefox十周年網路自由日系列活動『幸福的狐家元件』以及於華山文創產業園。

Firefox OS

Firefox OS 於 MWC 發表最新發展成果

  • 公佈 7 款全新 Firefox OS 行動裝置。
  • 全球 21 家電信商宣佈支援 Firefox OS。
  • 更高效能 FIREFOX OS 裝置問世；改寫入門級智慧手機定義。

首款 Firefox OS 開發手機 Flame 開賣

  • 為第一支 Firefox OS「參考平台手機」。
  • Flame具備中階手機的規格，主要針對開發者與貢獻者社群所設計。

2014 亞洲通訊博覽會 (MAE 2014)

  • 參與於上海舉行的MAE，與中華電信簽合作備忘錄，Open Web 陣營再下一城。
  • 宣佈首批平價 Firefox OS 智慧型手機即將於印度上巿。

年度媒體茶會

  • 首度揭露Firefox OS 研發中心重鎮與台灣分公司最新發展，吸引科技與產業線重要媒體踴躍出席。

首款 Firefox OS 電視棒現身

  • 新創公司 MathStick, 配合使用開放程式碼技術，開發出第一個 Firefox OS HDMI 電視棒。

Firefox OS 手機印度開賣

  • 首度進入亞洲巿場，鎖定智慧型手機首次使用者。
  • 改寫入門等級智慧型手機定義。

Firefox 進軍近 30 國

  • 中美洲已全境上巿，非洲即將入列。
  • 全球最親民價格的智慧型手機，已於菲律賓上巿。

Firefox 首款高階手機於日本上巿

  • Mozilla與KDDI 共同發表首款 Firefox OS 高階智慧手機「Fx0」。
  • 由知名設計師吉岡德仁 (Tokujin Yoshioka) 所設計，支援 4G LTE、NFC、WebRTC 等。

Community

COSCUP 開源人年會

  • 謀智台客編輯 gasolin 參與議程分享 “Firefox OS －大型開放原始碼專案是怎麼煉成的” 帶來熱烈回響！
  • 熱情的開源人參觀攤位，了解 Firefox OS 手機與 Mozilla；超過百位朋友訂閱 Firefox 電子報。

JavaScript 年度盛會 – JSDC 2014 研討會

  • Firefox OS 手機高人氣！「 Firefox 十週年 – 網路自由日」活動也創造高詢問度。
  • Mozilla Firefox OS 開發工程師 Greg 所主講議程 「Functional JavaScript, why or why not?」聽眾爆滿，現場座無虛席。

校園大使年中聚

  • 過往及現任的 Firefox 校園大使相見歡！
  • 分組拼豆活動、小任務與Appmaker 競賽，考驗創意與團隊默契。

美國小學生企業參訪

  • 分組拼豆活動、小任務與Appmaker 競賽，考驗創意與團隊默契。

摩茲工寮搬新家

  • 由社群共同管理的Mozilla空間。
  • 位於八德路，靠近華山藝文中心。

Open Web

Mozilla 與遊戲平台 Humble Bundle 合作

  • Mozilla 率先開發的 asm.js技術，讓開發者能提供高品質、免外掛的線上遊戲。
  • Firefox 使用者可直接體驗 8 款高人氣獨立製作遊戲。

W3C 宣佈 HTML5 進入建議階段

  • HTML5已列入 W3C 建議公告；也是成為 W3C 標準前的最後階段。
  • Mozilla為創新開發 HTML5 的組織之一，「建議」階段呼應了 Mozilla 十年來念茲在茲的使命與心血。

Blogs

謀智台客文章首度登上 Hacks

  • 謀智台客 John Hu 撰寫的” The Visibility Monitor supported by Gaia,” 首度登上Hacks。
