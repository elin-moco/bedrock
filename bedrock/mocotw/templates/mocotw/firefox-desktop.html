新分頁快速入口

{{ _('Next') }}

Firefox 在地化功能提供多樣化分頁選擇。透過新增設的多樣化分頁，使用者可以依照個人使用習慣去設定新分頁開啟的內容， 包括熱門網站、推薦網站、熱門造訪頁面。使用者也可以自訂個人化的熱門網站，開啟書籤將己加入書籤的連結拖到要放的九宮格內， 或在推薦網站裡，透過貼心的分類，快速找到並瀏覽你感興趣的網站或論壇。

網頁截圖

{{ _('Next') }}

使用者可以輕鬆透過網頁截圖的功能， 將完整的網頁轉成一個圖檔存放在桌面上，既方便又容易，相信會是你將網頁記錄成圖檔的絕妙好工具。

開關分頁設定

{{ _('Next') }}

對分頁做最精準的管理。使用者可以根據自己的使用習慣設定分頁的開啟關閉方式。 例如：點擊網頁連結或輸入網址時，可自訂想要另開新分頁或是直接在目前的分頁開啟。 若需要更精準的分頁管理，同樣在瀏覽器右下角，直接點選開關分頁設定，使用最下方的進階設定功能進行設定。
