歡迎加入 Firefox 校園大使！

此封電子郵件用以確認你已正式報名 Firefox 校園大使。接下來你可透過幾個方式向周遭的人宣揚 Mozilla 的理念：

1) 加入 Firefox OS 的 Team Mobilizer
你想搶先知道 Firefox OS 的最新消息嗎？也想和其他大使互動，協助創造 Mozilla 的歷史嗎？

Team Mobilizer 是由一群熱血的科技高手所組成，致力發揚 Mozilla 行動新產品 ─ Firefox OS。Mobilizer 將定期接到任務，以簡單方法向外宣達 Firefox OS 的理念。另外根據自己的所在地，Mobilizer 每個月均需於當地市場參加、協助、舉辦活動一次。 進一步了解自己所在地的 Team Mobilizer！

2) 看 Wiki 上的簡介
透過我們的 Wiki 頁面了解校園大使所應完成的任務、可利用的工具、相關活動等資訊。也可以在裡面找到你能參與的活動 (中文版說明請參考 Firefox 校園大使任務包)，以在校園中推廣 Mozilla 以及 Firefox (還能贏得獎品)。

3) 加入 Facebook 與 Twitter 的校園大使粉絲團！
到社交網站查看定期更新的資訊，並聯絡其他校園大使。

4) 找到自己學校的 Firefox 社團，還沒有社團的話就自己創社吧！
找到自己校內的大使，或者創社擔任校內第一位 Firefox 社長。

加入校園大使之後，每個月都會收到 Mozilla 所寄發的電子報，內含更新訊息、達成成就、Firefox 新聞，還有更多資訊。我們要和你隨時保持聯絡！

當然，只要你有任何疑問，也請你隨時反應給我們。

5) 與本地的 Firefox 校園大使交流互動
在台灣，已經有一群充滿熱忱的 Firefox 校園大使在為網路的開放與自由貢獻心力。想要認識他們嗎？歡迎加入校園大使 mailing list。

再次感謝你熱情參與 Mozilla 的行動，並努力協助我們打造更美好 Web 的終極使命。
